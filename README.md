# Development

node 0.10.x required.  Install from homebrew (macosx) or [ppa](https://launchpad.net/~chris-lea/+archive/node.js/) (ubuntu)

## Install prerequisites:

    % sudo gem install sass compass

## Start

    % npm i
    % npm start
